# 前端设计思路
* 主页 以youtube为蓝本，左侧固定导航栏，里面罗列功能，右侧为container，展示每个模块部分视频
* 选电影 照搬youtube
* 电影信息展示页面，排行榜，个性推荐，个人中心，登陆:以主页为蓝本，container部分显示电影的信息

* 登陆页面两种形式
	* 直接主页的下拉菜单
	* 单独一个页面
